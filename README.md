# Toan MBTI

A personalized MBTI-style quiz app built with Vite, React, TypeScript, Tailwind CSS, and shadcn/ui.

## Getting started

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
```

## Project notes

- Branding is centralized in `src/lib/brand.ts`.
- Update copy, app identity, and footer credit there first.
- The production output is generated in `dist/`.
